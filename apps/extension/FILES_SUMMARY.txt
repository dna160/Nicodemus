DEPLOYMENT READY
================

To get started:

1. Install dependencies:
   cd apps/extension && pnpm install

2. Configure environment:
   cp .env.example .env.local
   # Edit with Supabase credentials

3. Development:
   pnpm dev
   # Then load .plasmo/chrome-mv3-dev in Chrome

4. Production build:
   pnpm build
   # Creates .plasmo/chrome-mv3-prod

5. Package for Chrome Web Store:
   pnpm package
   # Creates extension.zip

All code is production-ready, fully typed, and copy-pasteable.
